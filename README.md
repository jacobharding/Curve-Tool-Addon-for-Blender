# Curve Tool Addon for Blender (2.93.x)
## Tutorial
- Open the search menu in the 3D view with 'f3' and type in CurveTool and select. The addon operator will then be active.
- Click on the surfaces you want the curve to go to, each click will be a single curve point. The 3D cursor will move to each click.
- Press enter or right click to draw the curve
- You can tweak the settings for the curve in the undo window in the bottom left of the 3D view:
    - Bevel Depth
    - Resolution
    - Curve type (POLY, BEZIER)
